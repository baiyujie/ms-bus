## For developer

1. 首先安装配置 node 环境, 需要 6.x 版本。

2. 全局安装构建工具 FIS3。
  ``` bash
  npm install fis3 -g
  ```
3. 克隆项目到本地，并安装依赖模块
  ``` bash
  git clone https://github.com/xxapp/ms-bus.git
  
  cd ms-bus
  
  npm install
  ```
4. 日常运行项目
  ``` bash
  
  npm run dev
  ```
